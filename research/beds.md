# Raised Bed Products Reference

## Compact Layout Configuration (RECOMMENDED)

The compact layout uses **12 standard 4x8 ft beds** - the most common and cost-effective size.

### Why 4x8 ft?
- **Standard size** - most products available, no extensions needed
- **Easy reach** - 2ft to center from either side
- **Best value** - cheapest per square foot
- **More beds** - better crop rotation flexibility

## How to Build 4x12 ft Beds

Since the largest standard bed is 4x8 ft, extend to 4x12 ft:

```
┌─────────────────────────────────────────────┐
│                  4x12 ft                     │
│  ┌──────────────────┐  ┌─────────────────┐  │
│  │   4x8 Base Kit   │  │ 4ft Extensions  │  │
│  │    (Vegega)      │  │   (2x 2ft)      │  │
│  └──────────────────┘  └─────────────────┘  │
└─────────────────────────────────────────────┘
```

**Components:**
- 1x Vegega 17" 4x8 (10-in-1) - $199.95
- 2x Vegega 17" 2ft extension panels - $131.90
- **Total: ~$332 per 4x12 bed**

## How to Build 8x12 ft Beds

Place two 4x12 sections side-by-side:

```
┌─────────────────────────────────────────────┐
│                  8x12 ft                     │
│  ┌─────────────────────────────────────┐    │
│  │           4x12 Section A             │    │
│  │   (4x8 base + 2x 2ft extensions)    │    │
```

## Back to Eden Fill Depths

For pure Back to Eden method:

| Bed Height | Soil Depth | Mulch Depth | Notes |
|------------|------------|-------------|-------|
| 12" | 8-10" | 2-4" | Adequate for most crops |
| 17" | 12-14" | 4-6" | **Recommended** - good depth + mulch |
| 32" | 24-28" | 4-6" | Extra deep for accessibility |

**Recommendation:** Use 17" beds with 12-14" compost/soil and 4-6" wood chip mulch.
